Bub Build focuses on engineering and research in **collaborative agents** and **human-agent co-design**.

- [bub-contrib: ecosystem packages and integrations](https://github.com/bubbuild/bub-contrib)

## Websites

## Focus

- Build agents that help people work together in real projects.
- Make agent behavior easy to understand, review, and hand off.
- Treat context construction as a product surface, not hidden runtime magic.
